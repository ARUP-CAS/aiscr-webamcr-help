## Kategorie návrhů

Pro orientaci — typy návrhů očekávané v tomto repozitáři:

- **Obsah** — aktualizace zastaralých informací, doplnění chybějících sekcí
- **Navigace** — zlepšení struktury `_quarto.yml`, reorganizace kapitol
- **Styly** — zlepšení CSS/SCSS, konzistence tématu
- **CI/CD** — optimalizace GitHub Actions workflows
- **Quarto konfigurace** — upgrades, nové funkce Quarto
- **Přístupnost** — alt texty obrázků, strukturální zlepšení
